public protocol WritableToFile: Writable

A type that can be used to write to a String file path.

    Parameter

    Parameter path: The path being written to.

    Parameter

    Parameter useAuxiliaryFile: If true, the data is written to an auxiliary file that is then renamed to the file. If false, the data is written to the file directly.
